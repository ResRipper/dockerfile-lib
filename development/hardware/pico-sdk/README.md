# Raspberry Pi - Pico SDK

RP MCU (RP2040/2350) build environment, containing both `pico-sdk` and `picotool`

- Base: Ubuntu 24.04
- SDK path (env: `PICO_SDK_PATH`): `/var/pico/pico-sdk`
- Included packages:
  - build-essential
  - ca-certificates
  - cmake
  - g++
  - gcc-arm-none-eabi
  - libnewlib-arm-none-eabi
  - libstdc++-arm-none-eabi-newlib
